Introduction
------------
This is a complete, fully functional, responsive and multipage personal portfolio website created by using HTML, CSS and JavaScript. The main goal of doing this website is prepare for my future to finding job easier. 

Description
-----------
This website is consists of 5 pages, which is Home, About, Services, Portfolio and Contact. The navigation bar is on the left side and consistent on all pages, user can switch to the page they want.
Besides that, there is a dark mode switcher on the top right of the website, user can switch to the mode they prefer. 

In the Home page, there is showed my name, some introduce and a "Download CV" button. when user click the button it will show my CV pdf, user can see and download it. I make the text design on the job description to make it looks nicer. 

In the About page, there is showed my details, such as my information, skills, education and work experience. Below my information, there is a "Hire me" button, after click it user will be on the contact page. 

In the Service page, there is showed some services that I offered.

In the Portfolio page, there is showed my recent projects.

In the Contact page, there is showed my contact information and also provided a form to let user send me email. I am using EmailJS as the email service, I already connected it to my gmail. When user successfully send the email, it will show 'Message sent âœ“' at the bottom of "Send button" and I can receive it at once. While user didn't fill up all the fields, it will show 'Write all the input fields. '.

Instruction
-----------
First, extract the zip file. 
Second, open the Visual Studio Code and open the folder extracted just now. 
Third, download the "open in browser" extension. 
Fourth, right click on the index.html, select "Open In Default Browser". 
After these steps done, the website can be running locally.
